# Express API Starter with Prisma and Typescript

A basic template for Express with TypeScript, Prisma and ZOD

## Setup

### Install Dependencies

```sh
npm install
```

### Create .env file

```sh
cp .env.sample .env
```

### Run Database

```sh
docker-compose up
```

### Migrate Database

```sh
npx prisma migrate dev
```

## Development

```sh
npm run dev
```

## Lint

```sh
npm run lint
```
